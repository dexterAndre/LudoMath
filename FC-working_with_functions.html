A function is a mechanism that takes input and returns one output.

The definition above is not mathematically rigid, but it will do for our purposes. It is important to note that a function can take in multiple inputs, but it can only return one output. If it returns several outputs, then it is not a function. To put this into perspective, let's define a circle in 2D. Given a center and a radius, the formula for a circle can be written as: $$(x - x_0)^2 + (y - y_0)^2 = r^2,$$ where \(x_0\) and \(y_0\) are the x- and y-coordinates of the center point, and \(r\) is the radius of the circle.

Interpreting Functions

A good way to build an intuition of how functions work and behave, is to plot their graph. We can easily let graphing software do this for us, and we will use GeoGebra for this - it's a decent piece of math software, and it's free. You can download the app for desktop or use it online. Personally, I like the GeoGebra Classic 5 version the best, probably because I've been using it for so long.
