Welcome to LudoMath, a website teaching mathematics for 3D game development!

This site is in pre-alpha stage, and its main functionalities are not in place yet. Expected alpha functionalities:

  • Basic site navigation
  • Interactive WebGL programs, for example a quaternion visualizer
  • Interactive problems, letting the user solve a problem and then check if you got it right

Following is a list of expected topics. These may change, and the site's content structure may also change.

  • Numbers & Algebra
  • Linear Algebra
  • Geometry
  • Functions & Curves
  • Discrete Mathematics
